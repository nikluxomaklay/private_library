# Task 004-editable-reading-log

## Общее описание:
Нужно добавить возможность редактировать Reading log.

## User story 1: Просмотр детальной информации о Reading log
**Как пользователь системы**, я хочу иметь возможность перейти на страницу просмотра детальной информации о Reading log.

### Места перехода на детальную страницу Reading log:
- Home - Период Reading log должен быть ссылкой на детальную страницу.
- Reading log список - Период Reading log должен быть ссылкой на детальную страницу.
- Book edition детальная страница - Период Reading log должен быть ссылкой на детальную страницу (Текущие ссылки на год чтения нужно убрать).

### Внешний вид детальной страницы Reading log:
- Аналогично странице создания Reading log, только read only.

## User story 2: Редактирование Reading log
**Как пользователь системы**, я хочу иметь возможность отредактировать Reading log.

### Переход на страницу редактирования:
- На детальной странице Reading log в блоке действий должна быть кнопка "Update" аналогично другим детальным страницам.

### Внешний вид страницы редактирования Reading log:
- Аналогично странице создания Reading log.

### Поля, доступные для редактирования:
- Year start
- Month start
- Year finish
- Month finish

## Дополнительная информация и условия:
- Для всех изменяемых полей есть возможность выбрать пустое значение.
