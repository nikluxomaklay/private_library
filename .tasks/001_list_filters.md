# 001-list-filters

## User Story 1: Фильтрация списков
**Как пользователь системы**, я хочу иметь возможность фильтровать данные на страницах со списками.

### Перечень страниц и их фильтров:
- Reading log:
  - Год 'from' и/или 'to' - выбор из селектора
  - Месяц 'from' и/или 'to' - выбор из селектора
  - Год + месяц 'from' и/или 'to' - выбор из селектора
  - Название книги - поиск по подстроке
  - ФИО автора - поиск по подстроке
  - Название издательства - поиск по подстроке
  - Год издания - точное совпадение
  - Название книжной серии - поиск по подстроке
- Authors:
  - ФИО - поиск по подстроке
- Books:
  - Название книги - поиск по подстроке
  - ФИО автора - поиск по подстроке
- Book editions:
  - Название книги - поиск по подстроке
  - ФИО автора - поиск по подстроке
  - Название издательства - поиск по подстроке
  - Год издания - точное совпадение
  - Название книжной серии - поиск по подстроке
- Publishers:
  - Название издательства - поиск по подстроке
- Book series:
  - Название книжной серии - поиск по подстроке

### Acceptance Criteria:
1. На перечисленные страницы добавлены указанные фильтры и только они.
2. Все фильтры работают без ошибок.
3. Стиль оформления инпутов фильтров не отличается от уже существующих инпутов на страницах добавления контента.
