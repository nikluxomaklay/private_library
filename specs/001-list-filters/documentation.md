# Использование фильтров в частной библиотеке

## Обзор

В этом руководстве объясняется, как использовать новые функции фильтрации, добавленные в приложение частной библиотеки. Теперь вы можете фильтровать записи в следующих списках:

1. Журнал чтения
2. Авторы
3. Книги
4. Издание книг
5. Издательства
6. Серии книг

## Как использовать фильтры

### Журнал чтения

На странице журнала чтения вы найдете следующие параметры фильтрации:

- **Год начала от/до**: Выберите диапазон лет начала чтения книги
- **Месяц начала от/до**: Выберите диапазон месяцев начала чтения книги
- **Название книги**: Поиск по названию книги (частичное совпадение)
- **Имя автора**: Поиск по имени автора (частичное совпадение)
- **Название издательства**: Поиск по названию издательства (частичное совпадение)
- **Год публикации**: Точный год публикации книги
- **Название серии книг**: Поиск по названию серии (частичное совпадение)

### Авторы

На странице авторов вы можете:

- **Имя автора**: Поиск по имени автора (частичное совпадение)

### Книги

На странице книг вы можете:

- **Название книги**: Поиск по названию книги (частичное совпадение)
- **Имя автора**: Поиск по имени автора (частичное совпадение)

### Издание книг

На странице изданий книг вы можете:

- **Название книги**: Поиск по названию книги (частичное совпадение)
- **Имя автора**: Поиск по имени автора (частичное совпадение)
- **Название издательства**: Поиск по названию издательства (частичное совпадение)
- **Год публикации**: Точный год публикации
- **Название серии книг**: Поиск по названию серии (частичное совпадение)

### Издательства

На странице издательств вы можете:

- **Название издательства**: Поиск по названию издательства (частичное совпадение)

### Серии книг

На странице серий книг вы можете:

- **Название серии**: Поиск по названию серии (частичное совпадение)

## Особенности

### Частичное совпадение

Все текстовые поля фильтрации поддерживают частичное совпадение (поиск по подстроке). Это означает, что если вы ищете "Django", будут найдены книги с названиями "Django for Beginners", "Advanced Django", и т.д.

### Очистка фильтров

Чтобы сбросить все примененные фильтры и вернуться к исходному списку, нажмите кнопку "Сбросить фильтры".

### Ограничения

- Все текстовые фильтры ограничены 255 символами
- Специальные символы обрабатываются безопасным образом для предотвращения инъекций

## Примеры использования

1. **Найти все книги Джона Смита**:
   - Перейдите на страницу "Книги"
   - Введите "John Smith" в поле "Имя автора"
   - Нажмите "Фильтровать"

2. **Найти книги, прочитанные в 2022 году**:
   - Перейдите на страницу "Журнал чтения"
   - Установите "Год начала от" в 2022 и "Год окончания до" в 2022
   - Нажмите "Фильтровать"

3. **Найти все издания определенного издательства**:
   - Перейдите на страницу "Издание книг"
   - Введите название издательства в поле "Название издательства"
   - Нажмите "Фильтровать"
